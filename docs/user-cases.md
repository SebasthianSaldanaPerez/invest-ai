# Use Cases

# UC-001 - Analyze a Stock

## Goal

Analyze a company before making an investment decision.

## Main Flow

1. The investor opens InvestAI.
2. Searches for a stock ticker.
3. Reviews the historical price chart.
4. Reviews historical statistics (maximum, minimum, average prices).
5. Reads recent news and sentiment analysis.
6. Reviews financial indicators.
7. Reviews financial statements.
8. Checks the Opportunity Score.
9. Reads the AI-generated analysis.
10. Decides whether to invest.
11. Optionally saves the stock to the Watchlist.

# UC-002 - Register a Purchase

## Goal

Register a new investment.

## Main Flow

1. Open Investment Journal.
2. Create a new operation.
3. Select the stock.
4. Enter purchase date.
5. Enter purchase price.
6. Save the Opportunity Score.
7. Add optional notes.
8. Save the operation.

---

# UC-003 - Register a Sale

## Goal

Complete an existing investment operation.

## Main Flow

1. Open Investment Journal.
2. Filter active operations.
3. Select an operation.
4. Enter sale date.
5. Enter sale price.
6. Update notes if necessary.
7. Save changes.

The system automatically calculates:

- Holding period
- Return
- Profit/Loss

# UC-004 - Analyze Investment Performance

## Goal

Analyze personal investment history.

## Main Flow

1. Open Personal Analytics.
2. View performance chart.
3. Review investment statistics.
4. Identify best-performing investments.
5. Identify worst-performing investments.
6. Review notes associated with each investment.
7. Request an AI summary of investment behavior.

---

# UC-005 - Configure Opportunity Score

## Goal

Customize the Opportunity Score calculation.

## Main Flow

1. Open Preferences.
2. Configure scoring rules.
3. Assign weights to indicators.
4. Save configuration.
5. Future analyses use the updated configuration.
