# User Stories

## US-001 - Analyze a Stock

As an investor, I want to search for a stock so I can analyze its financial information before making an investment decision.

---

## US-002 - Save a Stock to the Watchlist

As an investor, I want to save interesting stocks in a watchlist so I can analyze them later.

---

## US-003 - Register a Purchase

As an investor, I want to register my stock purchases so the system can learn from my investment decisions.

---

## US-004 - Register a Sale

As an investor, I want to register the sale of an investment so the system can calculate its performance.

---

## US-005 - Analyze My Investment History

As an investor, I want to review my investment history so I can identify successful and unsuccessful investment patterns.

---

## US-006 - Configure My Investment Strategy

As an investor, I want to configure my own evaluation rules so the Opportunity Score reflects my personal investment strategy.

---

## US-007 - Receive AI Assistance

As an investor, I want the AI to explain financial information and summarize relevant news so I can compare my own analysis with the AI's interpretation.

---

## US-008 - View Personal Performance

As an investor, I want to visualize my investment performance over time so I can evaluate my long-term decision making.
